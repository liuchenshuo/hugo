# NodeBB论坛，初始化后一些简单配置说明
管理员登录后，点击首页图标最后一个即可进入管理员界面
1. 设置语言为中文
    **SETTING -> Languages** 选择 简体中文，点击页面右下角的保存按钮进行保存，然后刷新界面
    可以看到已经变成中文格式
2. 修改网站主体：名称、图标等
